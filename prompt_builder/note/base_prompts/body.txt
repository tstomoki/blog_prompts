# note記事作成にあたっての必須アドバイス

noteで成功する副業系アカウントには以下の特徴があります。これらを十分考慮して記事を作成します。

- 一貫性と専門性: 特定のテーマに特化した専門的な情報を継続的に提供し、読者の信頼と共感を獲得する。

- 読者の悩み解決: 読者が抱える具体的な悩みや課題への解決策を提示する

- 体験談の共有: 成功体験だけでなく、失敗談や挫折経験も共有し、読者との親近感や信頼感を築く。

- クリックしたくなるタイトル: 一覧で表示された場合に見た人がたまらずクリックしてしまうようなタイトルにする

- 実践的なノウハウ提供: 具体的かつ実践可能な手順やコツを紹介する。

# 記事作成時に必ず守る条件

①HERMの法則への分類

記事の結論や得られた結果を、以下のHERMの法則に従い分類し、どれに該当するかを明示してください。

H：Health（健康、美容、容姿）

A：Ambition（将来、夢、キャリア）

R：Relation（人間関係、恋愛、結婚）

M：Money（お金）

# 有料記事を記載する際に意識すべきこと

「無料部分」だけでも十分な価値・説得力があり、「有料部分を読みたい」と期待させる構成にする

「無料」→「有料」の導線をスムーズに作る

「有料でも価値があった」と読者が満足する内容にする

課金前に「この記事で解決できる問題」と「解決できない問題」を明確に伝える

# 記事の読みやすさのためのルール

見出しは大見出しのみで4つ程度

結論を最初に書かない（最後まで読ませる工夫をする）

読者が記事を読む前に対象・範囲を明示

文章のリズムを意識し、続きを読みたくなるようにする

段落分けと改行を必ず行う（5行以上の文章が続くときは必ず改行を挿入）

段落は最長10行ごとに空白行を挟んで区切る

一文を長くしすぎない

本文は最低でも5000文字以上

改行は多めに入れることを意識する
特に句点のあとに改行をいれることで読みやすい文章になることを意識する

# ブログ記事のフレームワーク

以下のブログ記事のフレームワークの流れに従って文章を書いてください。

0. リード文
タイトルから1つ目の見出しの前にリード文として、「読んでいただいたお礼」を簡単に記載する。

1. 導入
最初に読者の興味を引く魅力的な導入部を書きます。
タイトルと関連性を持たせ、読者が記事を続けて読みたいと思わせる問題提起やストーリーを用意します。

2. 問題の説明
問題提起の後、読者に対して記事で解決しようとしている問題や課題を詳しく説明します。
読者がなぜこの情報が必要なのかを理解できるようにしましょう。

3. ソリューションの提示
本文の中で問題を解決するための具体的なソリューションや方法を提案します。
考えられるソリューションやノウハウを具体的なステップや戦略として説明します。

4. 実践やケーススタディ
ソリューションの理解を助けるために、実践の成功事例やケーススタディを挙げます。これにより読者に具体的なイメージを提供します。

5. ステップバイステップのガイダンス
ソリューションを実行するためのステップバイステップの手順やアクションアイテムを提供します。
読者が実践に移しやすくなります。

6. 注意事項やポイント
重要な注意事項やポイントを強調し、読者に成功への道を効果的に進むための指針を提供します。

7. 質問への回答
読者が持つかもしれない疑問や疑念に対して、明確な回答を提供します。読者が不安を解消し、信頼感を得られるようにします。

8. まとめ
本文の最後に重要なポイントをまとめ、再度読者に価値を提供します。解決策の要点や次のステップを強調します。

9. 行動の促進
読者に次のステップを促すための行動呼びかけを行います。
例えば、本noteアカウントの他の記事を読むことを呼びかけたり、ソーシャルメディアでのシェアやX(旧Twitter)アカウントのフォローをさせることなどを提案します。

10. 関連記事へのリンク
本アカウントで出している関連記事へのリンクを挿入し、本アカウント記事内で読者に関心を持ってもらえる他の記事に誘導します。

11. お礼

記事の最後には必ず「ここまで読んでいただきありがとう」というようなお礼を記載する

# 記事作成時の心得

- 有料部分を記載する際は、無料部分の内容をさらに掘り下げて具体的なノウハウや実践方法を提供する
- 読者が自身の課題や問題に応用できる具体的かつ価値ある情報を記載
- 有料購入した価値を十分感じられる実践的な内容にする
- 記事を読み終わった後に読者のモチベーションを高めるような(ネガティブな内容だけで終わらない)内容を意識する
- 本文中で文字のどこに力を入れて読むべきか、速読するときに有用なように大事なところは太字にしたり下線を引くなどの工夫を入れてください
